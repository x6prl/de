Irregular verbs (about 15%):

trinken = to drink

schreiben = to write

Perfekt

Perfekt = auxiliary verb (haben/sein) + Partizip II

Gestern habe ich lange gearbeitet.
= Yesterday I worked for a long time.

For irregular verbs, you usually have to learn the participle forms by heart.

Examples of Partizip II:
gearbeitet = worked
gegessen = eaten
getrunken = drunk
gefahren = gone / driven / traveled
gegeben = given
genommen = taken

Ich habe einen Kuchen genommen.
= I took a cake.

If a verb is irregular in English, it is often irregular in German too. This is not always true, but it can help.
